# PAGE_FLOW.md

# Jankalyan Page Flow

Version: 1.0

Status: 🔒 FROZEN

---

# Purpose

This document defines the complete page structure, navigation flow, permissions, and user journey for the Jankalyan platform.

All frontend and backend development must follow this page flow.

No page structure may be modified without explicit approval.

---

# Public Module

Accessible without login.

---

## 1. Home / Landing Page

Purpose

Introduce the platform and encourage citizens to report civic issues.

Sections

- Hero Banner
- Search Bar
- Complaint Statistics
  - Total Complaints
  - Resolved
  - Pending
- Recent Complaints
- OpenStreetMap Preview
- Categories
- Report Complaint CTA

Navigation

Home →
- Login
- Register
- Public Complaints
- Complaint Details

---

## 2. Login Page

Features

- Email
- Password
- Remember Me
- Login Button
- Register Link

Successful Login

Redirect based on role

- USER → Citizen Dashboard
- ADMIN → Admin Dashboard

---

## 3. Register Page

Features

- Full Name
- Email
- Mobile Number
- Password
- Confirm Password
- Register Button

Successful Registration

Redirect to Login Page

---

## 4. Public Complaints

Features

- List View
- Map View
- Search
- Filter by Category
- Filter by Status
- Sort
  - Newest
  - Most Upvoted

Only APPROVED complaints are visible.

---

## 5. Complaint Details

Features

- Complaint Images
- Title
- Description
- Category
- Location
- Status
- Vote Count
- Upvote Button
- Reported Date

Privacy

Reporter information is hidden from the public.

(Create & Edit)

## 8. My Complaints

Features

- Search
- Status Filter
- Edit
- Delete

Rules

Edit/Delete allowed only when status is SUBMITTED.

## 9. Profile

Features

- Update Name
- Update Mobile Number
- Change Password
- Profile Picture (Optional)

# User Journey

## Citizen Journey

Visitor
↓
Home Page
↓
Register
↓
Login
↓
Dashboard
↓
Create Complaint
↓
Select Category
↓
Write Description
↓
Upload Image
↓
Select Location
↓
Submit Complaint
↓
Track Complaint Status
↓
Logout

---

## Admin Journey

Admin Login
↓
Admin Dashboard
↓
View All Complaints
↓
Search / Filter Complaints
↓
Open Complaint Details
↓
Verify Complaint
↓
Approve / Reject Complaint
↓
Update Complaint Status
↓
Delete Spam Complaint (if required)
↓
Logout
