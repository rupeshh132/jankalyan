## Complaint Module

### Edit Complaint
Status: Planned

Reason:
The current backend does not expose an Update Complaint endpoint.

Future Work:
- Add PUT/PATCH /api/v1/complaints/{id}
- Allow owner to edit complaint
- Replace images
- Update location
- Update category
- Update description
- Maintain audit history

# Notes

These features are intentionally deferred to keep Jankalyan v1.0 stable, production-ready, and focused on the core civic complaint workflow.

Future implementations must preserve the existing architecture and follow the project's frozen API and coding standards unless a new major version explicitly requires architectural changes.
